# ReViT

A React boilerplate with TypeScript, TailwindCSS and Vite.

## Installation/Usage

You can start using this repo as a template or clone the repostory and start hacking

```bash
  git clone https://github.com/rabira-hierpa/revit
  cd vite
  npm install
```
